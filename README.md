## Environment Variables

## Run Locally

Go to the project directory

Install dependencies

```bash
  npm install
```

Configuration file in .env

```bash
  cp .env
```

Start automation testing with generate allure report

```bash
  npm run generate /source_file
```

Generate Allure report

```bash
  npm run report
```

Start Allure report

```bash
  npm run start
```
